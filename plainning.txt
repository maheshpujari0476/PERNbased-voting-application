Routes

User Authentication:

=> /signup : POST -> Create new user account

Voting:

User Profile:

=> /profile : GET -> Get the user's profile information
=> /profile?/password : PUT -> change user password

=> /candidate : POST -> Create new candidate
=> /candidate/:candidateID: PUT -> update an exixting candidate
=> /candidate/:candidateID : DELETE -> Delete a candidate from the list
